Maximum a posteriori estimate of contamination for ancient DNA and endogenous mitochondrial genome inference

Download this project as a tar.gz file

About.

Upon sequencing mitochondrial DNA from ancient samples and aligning the molecules to a reference, deamination of the molecules can interfere with consensus calling. For hominin samples, contamination from modern humans involved in the excavation and experimental process can complicate consensus calling of the endogenous (i.e. pertaining to the original individual). Also, quantifying this contamination is crucial to evaluate which samples are the most promising.

schmutzi is a Bayesian maximum a posteriori estimator of contamination for ancient hominin samples. It also calls the endogenous mitochondrial consensus. More precisely, schmutzi is a set of programs aimed at ancient DNA data that can :

  • estimate contamination based on deamination patterns alone
  • call a mitonchondrial consensus for the endogenous genome. This consensus calls takes into account contamination and deamination.
  • estimate mitonchondrial contamination using the endogenous calls, scan a database modern human set of putative contamination source. It can identify the most likely contaminant among the database records. An exhaustive set of human MTs is provided with the package.
